About This Project

Project Overview

The AI-Enhanced Intrusion Detection System (IDS) is a machine learning-based cybersecurity tool that detects and classifies network intrusions in real time. Built using Python, Flask, and scikit-learn, it analyzes network traffic features to identify potential threats.

ML Model Details
  • Algorithm: Random Forest Classifier
  • Dataset: NSL-KDD (Network Security Lab - KDD)
  • Features: 10 key network traffic attributes
  • Classes: Normal, DoS, Probe, R2L, U2R
  • Accuracy: ~96% on test data
